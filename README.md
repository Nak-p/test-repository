# Antigravity Demo Project

これはAntigravityによって自動生成・拡張されたデモプロジェクト（多機能ToDoリスト）です。

## ✨ 主な機能
- **タスク管理**:
    - **追加**: テキスト入力と「追加」ボタンで新しいタスクを作成。
    - **完了**: タスクをクリックして完了/未完了を切り替え。
    - **削除**: タスク右側の「×」ボタンで削除（アニメーション付き）。
    - **優先度**: 低・中・高の3段階を設定可能。色分け（緑・オレンジ・赤）で視覚化。
- **デザイン**:
    - **グラスモーフィズム**: すりガラスのようなモダンなUI。
    - **ダイナミック背景**: 時間経過でゆっくりと色が変化する背景アニメーション。
    - **進捗バー**: タスクの消化率をリアルタイムで可視化。
    - **完了エフェクト**: 全タスク完了時に祝福アニメーションを表示。

## 📂 含まれるファイル
- **index.html**: アプリケーションの構造（入力フォーム、リスト、優先度選択）
- **style.css**: デザイン定義（アニメーション、レスポンシブ対応、優先度スタイル）
- **script.js**: ロジック（タスク追加/削除、イベント委譲、進捗計算）

## 🚀 試し方
1. `index.html` をブラウザで開いてください。
2. **タスクの追加**: 入力欄に文字を入れ、優先度を選んで「追加」を押します。
3. **タスクの操作**:
    - クリックして完了にする。
    - 右側の「×」ボタンで削除する。
4. **完了の瞬間**: すべてのタスクを完了にして、エフェクトを楽しんでください。
